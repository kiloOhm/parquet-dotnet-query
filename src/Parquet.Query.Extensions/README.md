# Parquet.Query.Extensions

`kiloOhm.Parquet.Net.Query.Extensions` is a convenience umbrella package for `parquet-dotnet-query`.

It installs:

- `kiloOhm.Parquet.Net.Query`
- `kiloOhm.Parquet.Net.Query.Extensions.Writing`
- `kiloOhm.Parquet.Net.Query.Extensions.Indexing`
- `kiloOhm.Parquet.Net.Query.Extensions.Search`
- `kiloOhm.Parquet.Net.Query.Extensions.Pooling`

Use this package when you want the full query stack without choosing individual extensions up front.
